## Description
The Penetration Testing Toolkit is a Python-based cybersecurity project developed to perform multiple penetration testing tasks such as Port Scanning, Brute Force Attacks, Banner Grabbing, and Network Scanning. In this project, the socket library is used to establish network connections, colorama is used for colored terminal output, and requests is used for HTTP-based brute force attacks. Each module is independently designed and can be launched from a single main menu interface.

The project is created using Python along with the requests, paramiko, colorama, python-nmap, and scapy libraries. The main goal of this project is to provide a modular and easy-to-use penetration testing toolkit that automates common security assessment tasks. Penetration testing is a critical part of cybersecurity because it helps identify weaknesses in systems before malicious attackers can exploit them.

The program presents the user with a menu to select any module. The Port Scanner scans a target IP for open ports within a specified range. The Brute Forcer attempts to find valid passwords for HTTP login pages using a wordlist. The Banner Grabber connects to open ports and retrieves service banner information to identify running services. The Network Scanner scans an entire subnet to discover live hosts. After each operation, the results are displayed clearly in the terminal with color-coded output.

This project helped in understanding important concepts such as network security, socket programming, HTTP requests, brute force techniques, banner grabbing, network discovery, and cybersecurity basics. It also improved Python programming skills and knowledge of how automated tools are used in ethical hacking and penetration testing.
